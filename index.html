Usage

SVGs is a completely open-source icon set with icons crafted for web apps. SVGs was made to perfect your project design, style and animation of your svg the way you want.

Basic usage

To use an icon, just copy and paste it into your HTML code:

<svg id="heart" class="svgs" viewBox="0 0 512 512">
   <title>Heart</title>
   <path d="M256 448a32 32 0 01-18-5.57c-78.59-53.35-112.62-89.93-131.39-112.8-40-48.75-59.15-98.8-58.61-153C48.63 114.52 98.46 64 159.08 64c44.08 0 74.61 24.83 92.39 45.51a6 6 0 009.06 0C278.31 88.81 308.84 64 352.92 64c60.62 0 110.45 50.52 111.08 112.64.54 54.21-18.63 104.26-58.61 153-18.77 22.87-52.8 59.45-131.39 112.8a32 32 0 01-18 5.56z" stroke-linecap="round" stroke-linejoin="round" class="svgs-fill-none svgs-stroke-width"></path>
</svg>

Variants

Each app icon in SVGs has a filled, outline and sharp variant. These different variants are provided to make your app feel native to a variety of platforms. The filled variant uses the default name without a suffix. Note: Logo icons do not have outline or sharp variants.

<svg id="heart">...</svg> <!--filled-->
<svg id="heart-outline">...</svg> <!--outline-->
<svg id="heart-sharp">...</svg> <!--sharp-->

Size

To specify the icon size, you can use the width and height attributes:

<svg width="32" height="32">...</svg>
<svg width="32" height="32">...</svg>

Or you can set a specific size by applying the CSS width and height properties on the svg component. It's recommended to use pixel sizes that are a multiple of 8 (8, 16, 32, 64, etc.)

svg {
   width: 32px;
   height: 32px;
}

Color / Fill

Specify the icon color by applying the color and/or fill CSS properties on the path component of the svg.

svg path {
   color: #FF6EF5;
}
(<svg id="heart">...</svg>) 
svg path {
   fill: #FF6EF5;
}
(<svg id="heart-outline">...</svg>)

Stroke weight

When using an outline icon variant it is possible to adjust the stroke weight, for improved visual balance relative to the icon's size or relative to the weight of adjacent text. You can set a specific size by applying the .svgs-stroke-width CSS custom property to the path component of the svg.

<svg id="infinite-outline" class="svgs" viewBox="0 0 512 512">
   <title>Infinite</title>
   <path d="M256 256s-48-96-126-96c-54.12 0-98 43-98 96s43.88 96 98 96c37.51 0 71-22.41 94-48M256 256s48 96 126 96c54.12 0 98-43 98-96s-43.88-96-98-96c-37.51 0-71 22.41-94 48" stroke-linecap="round" stroke-miterlimit="10" class="svgs-fill-none svgs-stroke-width"></path>
</svg>
svg path.svgs-stroke-width {
   stroke-width: 16px;
}
(<svg id="heart-outline">...</svg>)
